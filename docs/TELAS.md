# Mockups and Structure for All Screens

## Authentication Screens

### Login Screen
- **Inputs:** 
  - Email
  - Password
- **Buttons:** 
  - Login
  - Forgot Password
  - Register

### Registration Screen (Cadastro)
- **Inputs:** 
  - Name
  - Email
  - Password
  - Confirm Password
- **Buttons:** 
  - Register
  - Back to Login

### Password Recovery Screen (Recuperar Senha)
- **Inputs:** 
  - Email
- **Buttons:** 
  - Send Recovery Email
  - Back to Login

---

## Aluna Screens

### Home Screen
- **Components:** 
  - Welcome Message 
  - Upcoming Classes
  - Quick Links to other features

### Schedule Class Screen (Agendar Aula)
- **Inputs:** 
  - Date
  - Time
  - Class Type
- **Buttons:** 
  - Confirm
  - Cancel

### My Classes Screen (Minhas Aulas)
- **Components:** 
  - List of enrolled classes with details
  - Option to reschedule or cancel

### My Plan Screen (Meu Plano)
- **Components:** 
  - Subscription details
  - Payment information

### Events Screen
- **Components:** 
  - Upcoming events with RSVP options

### Profile Screen
- **Components:** 
  - Personal information (editable)
  - Change Password option

---
